Introducción a la programación

Paso 1. Conoce el Contexto.

Primero debemos entender que esto no es fácil, necesitamos tiempo, con una enorme curiosidad por las cosas. Tampoco existe la manera "correcta" de aprender a programar, cada persona tiene métodos, maneras, formas distintas, como cientos de caminos distintos, asi que no te limites a solo una idea.
El siguiente método que mostraremos sera el que nosotros vemos mejor para entender este campo.

Paso 2. Aprende la Lógica.

Antes de iniciar de verdad, necesitamos entender como funciona las cosas de una manera sistemática. A lo que le llamamos como un "algoritmo" donde surge un procedimiento para realizar una tarea o un objetivo pero con pasos detallados para llegar a una resolución del problema.
Recuerda que programar es como explicarle a una persona que no conoce nada, tienes que ir paso por paso.
Entiende la lógica: https://www.youtube.com/watch?v=D38d8SCtYqE&list=PLWtYZ2ejMVJlACGFXj4zpJ8cWdSjF05tm

Paso 3. Encuentra tu mision u objetivo.

La programacion es una habilidad poderosa actualmente, donde es usada para resolver problemas complejos y hacerlos mas simples. Que excelente forma de aprenderla con una motivacion en mente, ya sea automatizar un proceso pequeño o grande. Muchas de las razones por que sentimos que no entendemos algo es por la falta de curiosidad o motivacion.

Busca un problema simple e intenta resolverlo, encuentra situaciones reales como lo son: crear una extension del navegador para guardar enlaces de paginas utiles, crear una aplicacion web que pueda encontrar productos de diferentes tiendas online sin necesidad de visitar una por una, hasta lo que es administrar productos de una empresa. La motivacion es un impulso valioso a la hora de comprender la programacion
